# 🚀 React Admin Dashboard

A modern and fully responsive **React Admin Dashboard** built using **Material UI**, **Data Grid**, and powerful charting libraries.
This project demonstrates a complete dashboard system with **light & dark mode**, analytics, and rich UI components.

---

## 🌐 Live Demo

👉 https://dashboard-mui-eta.vercel.app/

---

## ✨ Features

* 🎨 Light & Dark Mode (Custom Theme System)
* 📊 Interactive Charts (Bar, Line, Pie, Geo)
* 📅 Calendar Integration (FullCalendar)
* 📋 Advanced Data Grid (Sorting, Filtering, Pagination)
* 📂 Sidebar Navigation (Collapsible)
* 📝 Forms with Validation (Formik + Yup)
* 📱 Fully Responsive Design
* ⚡ Fast Performance (Vite)

---

## 🛠️ Tech Stack

### Frontend

* React 19
* Material UI (MUI)
* React Router DOM

### UI & Visualization

* MUI Data Grid
* Nivo Charts (Bar, Line, Pie, Geo)
* FullCalendar

### Form Handling

* Formik
* Yup

### Build Tool

* Vite

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/dashboard.git
cd dashboard
```

Install dependencies:

```bash
npm install
```

---

## 🚀 Run Locally

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
